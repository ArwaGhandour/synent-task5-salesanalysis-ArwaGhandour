 Superstore Sales Data Analysis 📈

 Project Overview
This project is part of the Synent Technologies Internship Program (Task 5). The goal was to conduct a comprehensive **Sales Data Analysis** on a retail dataset to identify revenue trends, top-performing products, and market segments.

 Dataset Description
The dataset used is the **Superstore Sales Dataset**, which contains retail transaction data from 2014 to 2017. Key features include:
* **Order Date:** Used for time-series analysis.
* **Category & Sub-Category:** Used to identify high-revenue product groups.
* **Segment:** Analyzing consumer vs. corporate sales.
* **Region:** Identifying geographical sales distribution.


 Tech Stack
* **Python** 
* **Pandas:** Data cleaning and aggregation.
* **Matplotlib & Seaborn:** Advanced data visualization.
* **Google Colab:** Development environment.



 Key Insights
Based on the visual analysis, here are the main findings:
1. **Seasonal Trends:** Sales show a consistent upward trend every year, with a significant **peak in December**, likely due to holiday shopping.
2. **Category Leadership:** The **Technology** category generates the highest total revenue compared to Furniture and Office Supplies.
3. **Market Segmentation:** The **Consumer segment** is the primary driver of sales, accounting for over **50%** of total revenue.
4. **Top Products:** High-value technology items (like Phones and Copiers) are the top 10 revenue-generating products.



 Repository Structure
